# 🎓 Интерактивный тест для учеников

Полноценное веб-приложение для проведения тестирования с красивыми анимациями, таймером, защитой от повторного прохождения и панелью учителя.

## ✨ Возможности

- **30 случайных вопросов** из базы (или все, если меньше 30)
- **Перемешивание вопросов** — каждый раз разный порядок
- **Перемешивание ответов** — варианты в случайном порядке
- **Таймер 30 минут** с визуальными предупреждениями
- **Защита от повторного прохождения** — блокировка на 24 часа
- **Красивые анимации** — slide, fade, pulse, confetti, floating shapes
- **Панель учителя** с паролем, статистикой, поиском и экспортом в CSV
- **Сохранение результатов** в Google Sheets
- **Адаптивный дизайн** — работает на телефонах

## 🚀 Быстрый старт

### Шаг 1: Создайте Google Sheets

1. Откройте [Google Sheets](https://sheets.new)
2. Создайте лист **"Questions"** с такими заголовками:

| ID | Question | Option1 | Option2 | Option3 | Option4 | Correct | Active |
|----|----------|---------|---------|---------|---------|---------|--------|
| 1 | Сколько будет 2+2? | 3 | 4 | 5 | 6 | 2 | TRUE |
| 2 | Столица Франции? | Лондон | Берлин | Париж | Мадрид | 3 | TRUE |

3. Создайте лист **"Results"** с заголовками:

| Timestamp | Name | Score | Total | Percent | TimeSpent | Answers | Questions |

4. Скопируйте **ID таблицы** из URL (часть между `/d/` и `/edit`)

### Шаг 2: Настройте Google Apps Script

1. В таблице: **Расширения → Apps Script**
2. Удалите весь код по умолчанию
3. Вставьте код из `Code.gs`
4. Замените `ВАШ_ID_ГУГЛ_ТАБЛИЦЫ` на реальный ID таблицы
5. Сохраните (Ctrl+S)
6. Нажмите **Развернуть → Новое развертывание**
7. Тип: **Веб-приложение**
8. Кто имеет доступ: **Все**
9. Нажмите **Развернуть** и подтвердите разрешения
10. Скопируйте **URL веб-приложения**

### Шаг 3: Обновите код

В `index.html` и `teacher.html` найдите:
```javascript
GOOGLE_SCRIPT_URL: 'https://script.google.com/macros/s/ВАШ_SCRIPT_ID/exec'
```

Замените на ваш реальный URL.

Также в `teacher.html` измените пароль:
```javascript
TEACHER_PASSWORD: 'teacher123'
```

### Шаг 4: Загрузите на GitHub Pages

```bash
# Создайте репозиторий на GitHub
git init
git add index.html teacher.html README.md
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/ВАШ_НИК/ВАШ_РЕПО.git
git push -u origin main
```

В настройках репозитория: **Settings → Pages → Source → Deploy from a branch → main / root**

Ваш тест будет доступен по адресу:
```
https://ваш-ник.github.io/ваш-репо/
```

## 📋 Формат вопросов в Google Sheets

| Поле | Описание |
|------|----------|
| ID | Уникальный номер вопроса |
| Question | Текст вопроса |
| Option1-4 | Варианты ответов |
| Correct | Номер правильного ответа (1-4) |
| Active | TRUE/FALSE — показывать ли вопрос |

## 🔒 Безопасность

- Пароль панели учителя хранится в открытом виде в JS (для GitHub Pages это нормально)
- Для production используйте backend с аутентификацией
- Блокировка повторного прохождения через localStorage (можно обойти, но защищает от случайных повторов)

## 🎨 Кастомизация

### Изменить количество вопросов:
```javascript
QUESTIONS_COUNT: 30  // в index.html
```

### Изменить время:
```javascript
TIME_MINUTES: 30  // в index.html
```

### Изменить время блокировки:
```javascript
BLOCK_HOURS: 24  // в index.html
```

## 📱 Демо-режим

Если Google Sheets не подключен, приложение автоматически переключается на демо-вопросы, чтобы вы могли протестировать интерфейс.
